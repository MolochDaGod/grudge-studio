⚔️ Game Data JSON

Fetched by API Server at /api/v1/game-data

  • weapons.json — weapon definitions + stats
  • armor.json — armor sets by type
  • materials.json — crafting materials
  • recipes.json — crafting recipes
  • Additional datasets as added

🖼️ Sprite Icons

48×48 and 96×96 weapon/armor icons

  • icon-index.json — master icon manifest
  • icons/weapons/*.png — per-weapon sprite
  • icons/armor/*.png — per-armor sprite
  • Referenced by grudachain app for card rendering

🔄 API Server Caching

In-memory cache with stale-while-revalidate

  • Cache-Control: public, max-age=300 (5 min)
  • X-Cache-Status header (HIT / STALE)
  • POST /api/v1/game-data/cache/clear to force refresh
  • Cross-resource search via /api/v1/game-data/search
