# Puter File System (FS) API

Cloud storage operations for GRUDGE Warlords using Puter.js.

## Overview

The Puter FS API provides cloud file storage without infrastructure setup. With the User-Pays Model, users cover their own storage and bandwidth costs.

## Best Practices for GRUDGE

1. **Use meaningful paths**: Organize files by category (`sprites/`, `configs/`, `saves/`)
2. **Handle errors gracefully**: Always wrap FS operations in try/catch
3. **Use async/await**: All FS operations are asynchronous
4. **Dedupe uploads**: Use `dedupeName: true` to avoid filename conflicts

## Core Functions

### Write File
```javascript
// Write text file
await puter.fs.write('saves/character.json', JSON.stringify(characterData));

// Write with deduplication (auto-rename if exists)
await puter.fs.write('sprites/hero.png', imageBlob, { dedupeName: true });

// Create parent directories automatically
await puter.fs.write('deep/nested/path/file.txt', 'content', { createMissingParents: true });
```

### Read File
```javascript
// Read file as blob
const blob = await puter.fs.read('saves/character.json');
const content = await blob.text();
const data = JSON.parse(content);

// Read binary file
const imageBlob = await puter.fs.read('sprites/hero.png');
const imageUrl = URL.createObjectURL(imageBlob);
```

### Create Directory
```javascript
// Create single directory
await puter.fs.mkdir('sprites');

// Create with missing parents
await puter.fs.mkdir('assets/sprites/characters', { createMissingParents: true });

// With deduplication
await puter.fs.mkdir('backup', { dedupeName: true });
```

### List Directory
```javascript
// List all items in directory
const items = await puter.fs.readdir('sprites');
items.forEach(item => {
    console.log(`${item.name} - ${item.is_dir ? 'Directory' : 'File'} - ${item.size} bytes`);
});

// List root directory
const rootItems = await puter.fs.readdir('./');
```

### File Operations
```javascript
// Rename
await puter.fs.rename('old_name.txt', 'new_name.txt');

// Copy file to directory
await puter.fs.copy('source.txt', 'destination_folder/');

// Move file
await puter.fs.move('source.txt', 'destination_folder/');

// Delete file or directory
await puter.fs.delete('unwanted_file.txt');
```

### Get File Info
```javascript
const info = await puter.fs.stat('character.json');
console.log(`Name: ${info.name}`);
console.log(`Path: ${info.path}`);
console.log(`Size: ${info.size} bytes`);
console.log(`Created: ${info.created}`);
console.log(`Is Directory: ${info.is_dir}`);
```

### Upload from File Input
```javascript
const fileInput = document.getElementById('file-input');
fileInput.onchange = async () => {
    const file = await puter.fs.upload(fileInput.files);
    console.log(`Uploaded to: ${file.path}`);
};
```

## GRUDGE Integration Examples

### Save Character Progress
```javascript
async function saveCharacter(characterId, data) {
    const path = `grudge/saves/${characterId}.json`;
    await puter.fs.write(path, JSON.stringify(data), { createMissingParents: true });
}
```

### Load Character Progress
```javascript
async function loadCharacter(characterId) {
    try {
        const path = `grudge/saves/${characterId}.json`;
        const blob = await puter.fs.read(path);
        return JSON.parse(await blob.text());
    } catch (e) {
        return null; // No save found
    }
}
```

### Upload Sprite Sheet
```javascript
async function uploadSprite(file, category) {
    const path = `grudge/sprites/${category}/${file.name}`;
    await puter.fs.write(path, file, { dedupeName: true, createMissingParents: true });
}
```

## Error Handling

```javascript
try {
    await puter.fs.read('nonexistent.txt');
} catch (error) {
    if (error.code === 'subject_does_not_exist') {
        console.log('File not found');
    } else {
        console.error('FS Error:', error.message);
    }
}
```
