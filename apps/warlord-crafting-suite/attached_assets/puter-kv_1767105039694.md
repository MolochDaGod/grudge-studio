# Puter Key-Value Store API

Persist application data using key-value pairs in the cloud.

## Overview

The KV Store provides fast, reliable key-value storage without server setup. Perfect for:
- User preferences and settings
- Session data and state
- Counters and statistics
- Cached data

## Best Practices for GRUDGE

1. **Use prefixes**: Namespace keys to avoid collisions (`grudge_`, `character_`, etc.)
2. **Store JSON**: Serialize complex objects with `JSON.stringify()`
3. **Handle missing keys**: Always check for null/undefined values
4. **Use incr/decr for counters**: Atomic operations prevent race conditions

## Core Functions

### Set Value
```javascript
// Simple string
await puter.kv.set('username', 'GrudgeWarrior');

// Number
await puter.kv.set('gold', 1000);

// Boolean
await puter.kv.set('premium', true);

// Complex object (must stringify)
await puter.kv.set('character_data', JSON.stringify({
    name: 'Hero',
    level: 10,
    class: 'Miner'
}));
```

### Get Value
```javascript
// Get string
const username = await puter.kv.get('username');

// Get and parse JSON
const dataStr = await puter.kv.get('character_data');
const data = dataStr ? JSON.parse(dataStr) : null;

// With default value
const gold = (await puter.kv.get('gold')) || 0;
```

### Increment / Decrement
```javascript
// Increment counter (atomic)
const newCount = await puter.kv.incr('login_count');

// Decrement counter (atomic)
const remaining = await puter.kv.decr('lives');

// Increment by amount (call multiple times or use set)
await puter.kv.set('gold', (await puter.kv.get('gold') || 0) + 100);
```

### Delete Key
```javascript
// Delete single key
await puter.kv.del('temporary_data');

// Delete multiple keys
await Promise.all([
    puter.kv.del('key1'),
    puter.kv.del('key2'),
    puter.kv.del('key3')
]);
```

### List Keys
```javascript
// List all keys
const keys = await puter.kv.list();

// List with pattern matching (glob)
const characterKeys = await puter.kv.list('character_*');

// List with values
const keyValues = await puter.kv.list(true);
keyValues.forEach(kv => {
    console.log(`${kv.key} = ${kv.value}`);
});

// List with pattern and values
const items = await puter.kv.list('grudge_*', true);
```

### Flush All Data
```javascript
// Clear ALL keys for this app (use with caution!)
await puter.kv.flush();
```

## GRUDGE Integration Examples

### GrudgeAccount Linking
```javascript
const LINKED_ACCOUNT_PREFIX = 'grudge_linked_account_';

async function linkGrudgeAccount(puterId, grudgeAccountId, username) {
    await puter.kv.set(`${LINKED_ACCOUNT_PREFIX}${puterId}`, JSON.stringify({
        id: grudgeAccountId,
        username: username,
        linkedAt: new Date().toISOString()
    }));
}

async function getLinkedAccount(puterId) {
    const data = await puter.kv.get(`${LINKED_ACCOUNT_PREFIX}${puterId}`);
    return data ? JSON.parse(data) : null;
}
```

### User Settings
```javascript
async function saveSettings(userId, settings) {
    await puter.kv.set(`settings_${userId}`, JSON.stringify(settings));
}

async function loadSettings(userId) {
    const data = await puter.kv.get(`settings_${userId}`);
    return data ? JSON.parse(data) : {
        sound: true,
        music: true,
        notifications: true
    };
}
```

### Session Management
```javascript
async function createSession(userId, token) {
    await puter.kv.set(`session_${token}`, JSON.stringify({
        userId,
        createdAt: Date.now(),
        expiresAt: Date.now() + (24 * 60 * 60 * 1000) // 24 hours
    }));
}

async function validateSession(token) {
    const data = await puter.kv.get(`session_${token}`);
    if (!data) return null;
    
    const session = JSON.parse(data);
    if (Date.now() > session.expiresAt) {
        await puter.kv.del(`session_${token}`);
        return null;
    }
    return session;
}
```

### Statistics Tracking
```javascript
async function trackEvent(eventName) {
    await puter.kv.incr(`stats_${eventName}`);
}

async function getStats() {
    const keys = await puter.kv.list('stats_*', true);
    return keys.reduce((acc, kv) => {
        const eventName = kv.key.replace('stats_', '');
        acc[eventName] = parseInt(kv.value) || 0;
        return acc;
    }, {});
}
```

## Error Handling

```javascript
try {
    await puter.kv.set('key', 'value');
} catch (error) {
    console.error('KV Error:', error.message);
    // Fallback to localStorage
    localStorage.setItem('key', 'value');
}
```
