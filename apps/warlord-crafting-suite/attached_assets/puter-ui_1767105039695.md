# Puter UI API

Create rich interfaces and interact with the Puter desktop environment.

## Overview

The UI API provides desktop integration features when running as a Puter app:
- Window management
- Dialogs and alerts
- File pickers
- System integration

## Authentication

### Authenticate with Puter
```javascript
// Request user authentication
await puter.ui.authenticateWithPuter();

// Check if signed in
if (puter.auth.isSignedIn()) {
    const user = await puter.auth.getUser();
    console.log(`Welcome, ${user.username}!`);
}
```

## Dialogs and Alerts

### Alert Dialog
```javascript
// Simple alert
await puter.ui.alert('Operation completed!');

// With title
await puter.ui.alert('Your character has been saved.', { title: 'Success' });
```

### Prompt Dialog
```javascript
// Get user input
const name = await puter.ui.prompt('Enter your character name:');
if (name) {
    console.log(`Character name: ${name}`);
}

// With default value
const gold = await puter.ui.prompt('Enter amount:', { defaultValue: '100' });
```

## Window Management

### Create Window
```javascript
const window = await puter.ui.createWindow({
    url: 'https://example.com',
    title: 'Game Window',
    width: 800,
    height: 600
});
```

### Window Properties
```javascript
// Set title
await puter.ui.setWindowTitle('GRUDGE Warlords - Character');

// Set size
await puter.ui.setWindowSize(1024, 768);
await puter.ui.setWindowWidth(1024);
await puter.ui.setWindowHeight(768);

// Set position
await puter.ui.setWindowPosition(100, 100);
await puter.ui.setWindowX(100);
await puter.ui.setWindowY(100);
```

## File Pickers

### Open File
```javascript
// Single file
const file = await puter.ui.showOpenFilePicker();
if (file) {
    const content = await file.read();
    console.log('File content:', await content.text());
}

// Multiple files
const files = await puter.ui.showOpenFilePicker({ multiple: true });

// Filter by type
const images = await puter.ui.showOpenFilePicker({
    accept: '.png,.jpg,.gif'
});
```

### Save File
```javascript
const file = await puter.ui.showSaveFilePicker({
    suggestedName: 'character.json'
});
if (file) {
    await file.write(JSON.stringify(characterData));
}
```

### Directory Picker
```javascript
const directory = await puter.ui.showDirectoryPicker();
if (directory) {
    console.log('Selected:', directory.path);
}
```

## System Integration

### Launch App
```javascript
// Launch another Puter app
await puter.ui.launchApp('code');

// Launch with arguments
await puter.ui.launchApp('editor', { file: '/path/to/file.txt' });
```

### Get Parent App
```javascript
const parent = await puter.ui.parentApp();
if (parent) {
    console.log('Launched from:', parent.name);
}
```

### Exit Application
```javascript
// Clean exit
await puter.ui.exit();

// Exit with status
await puter.ui.exit(0);
```

### Menu Bar
```javascript
await puter.ui.setMenubar([
    {
        label: 'File',
        items: [
            { label: 'New', action: () => createNew() },
            { label: 'Save', action: () => save() },
            { type: 'separator' },
            { label: 'Exit', action: () => puter.ui.exit() }
        ]
    },
    {
        label: 'Help',
        items: [
            { label: 'About', action: () => showAbout() }
        ]
    }
]);
```

### Get Language
```javascript
const lang = await puter.ui.getLanguage();
console.log('Current language:', lang); // e.g., 'en', 'es', 'fr'
```

## Event Handling

### Window Events
```javascript
puter.ui.on('windowResize', (width, height) => {
    console.log(`Window resized to ${width}x${height}`);
});

puter.ui.onWindowClose(() => {
    // Save state before closing
    saveState();
    return true; // Allow close
});
```

### Launch Events
```javascript
// Handle launch with files
puter.ui.onLaunchedWithItems((items) => {
    items.forEach(item => {
        console.log('Opened with:', item.path);
    });
});

// Check if launched with items
if (await puter.ui.wasLaunchedWithItems()) {
    // Handle initial items
}
```

## Additional UI

### Spinners
```javascript
// Show loading
puter.ui.showSpinner();

// Do work...
await longOperation();

// Hide loading
puter.ui.hideSpinner();
```

### Color Picker
```javascript
const color = await puter.ui.showColorPicker();
if (color) {
    console.log('Selected color:', color); // e.g., '#ff5500'
}
```

### Font Picker
```javascript
const font = await puter.ui.showFontPicker();
if (font) {
    console.log('Selected font:', font.family, font.size);
}
```

### Social Share
```javascript
await puter.ui.socialShare({
    title: 'Check out my character!',
    text: 'My GRUDGE Warlords character reached level 50!',
    url: 'https://grudge-warlords.puter.site'
});
```

## Best Practices

1. **Check Puter environment**: Some UI features only work in Puter desktop
2. **Handle cancellation**: File pickers return null if cancelled
3. **Save before exit**: Use onWindowClose to save state
4. **Use appropriate dialogs**: Alert for info, prompt for input
5. **Respect user preferences**: Check getLanguage() for localization
