API Endpoints

GET /api/health
Health check & server status
POST /api/auth/register
Create new account (username, password, email?, puterId?)
POST /api/auth/login
Login with credentials
POST /api/auth/token
Generate SSO token for linked Puter account
POST /api/auth/validate
Validate authentication token
GET /api/auth/puter/:puterId
Lookup account by Puter UUID
GET /api/sync/status
Storage & sync status
